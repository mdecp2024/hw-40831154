w15

從 1 累加到 100

addto

def addto(start, end):

定義一個名為 addto 的函數，它有兩個參數：start 和 end，分別表示累加的起始值和結束值。

        total += i

在每次迴圈中，將變數 i（目前的數字）加到 total 上，這樣就會逐步累加起來。

    return total

當所有迴圈執行完畢後，返回 total，這就是從 start 到 end 所有整數的總和。

add_only_even

       if i % 2 == 0:

在迴圈中檢查每個數字 i 是否為偶數。使用 i % 2 == 0 來檢查 i 除以 2 的餘數是否為 0，即是偶數。

result = add_only_even(1, 100)

呼叫 add_only_even 函式，並傳入 1 和 100 作為參數，計算從 1 到 100 的所有偶數的總和，並將結果儲存到變數 result。

add_avoid_8

        if '8' in str(i):
            continue

這一行檢查當前的數字 i 是否包含數字 '8'。首先將 i 轉換成字串（str(i)），然後檢查字串中是否含有字符 '8'。如果包含 '8'，則使用 continue 跳過當前迴圈，跳到下一個數字，這樣就不會將此數字加到總和中。
